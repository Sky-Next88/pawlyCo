# pawlyCo 🐾
Pet's Happy Home —— A one-stop service platform for pets and their owners.

## Project Overview
pawlyCo is a pet-focused website dedicated to health, supplies, and community, aiming to help every furry friend live a happy life.

## Key Features
- 🐱 Pet health record management
- 🛒 Curated pet product store
- 💬 Pet owner community
- 📅 Vaccine and deworming reminders

## Tech Stack
- HTML5 / CSS3 / JavaScript
- Deployed on GitHub Pages
